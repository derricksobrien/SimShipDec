## Quick Start

To view the website locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/derri/derri.github.io.git
   ```

2. Open `index.html` in your web browser

3. Or use a local server:
   ```bash
   python -m http.server 8000
   # or
   npx http-server
   ```

4. Navigate to `http://localhost:8000`

## Customization

To customize the website:

1. **Change colors**: Edit CSS variables in `styles.css` (lines 7-16)
2. **Update content**: Edit text and sections in `index.html`
3. **Modify layout**: Adjust grid and flexbox properties in `styles.css`
4. **Add sections**: Copy existing section blocks and customize
